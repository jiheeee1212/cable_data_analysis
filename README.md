# 케이블 데이터 고객 분석 프로젝트

## 1. 프로젝트 개요
본 프로젝트는 케이블 데이터를 기반으로 한 **샘플 데이터**를 사용하여
고객 행동 분석을 수행하고, 해지 고객의 특성 파악 및 인사이트 도출을 목표로 합니다.

## 2. 프로젝트 목표
- 고객 시청 로그와 VOD 콘텐츠 데이터를 결합한 분석 데이터셋 구축
- 데이터 무결성 검증을 통한 분석 신뢰성 확보
- 해지 고객과 유지 고객 간의 특성 차이 분석
- 향후 해지 예측 모델링을 위한 기초 데이터 준비
